## Layout Principles

### Global

- wide desktop margins
- clean header
- minimal navigation
- strong type hierarchy
- large empty space used intentionally
- thin dividing lines
- compact but readable metadata blocks
- no generic card-heavy layout unless explicitly part of the design

### Home

Recommended order:

{{HOME_SECTIONS}}

The Home page should quickly move visitors toward primary content.

### Detail Pages

Detail pages should prioritize title, metadata, primary content, related entries, credits or notes, external links, and media references.

### Media / Archive Pages

Use an archive-like structure for media entries. If content is not available, omit the entry or use a polished empty state. Do not fake content.
